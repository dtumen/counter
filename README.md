# Проект счётчик - **IT-incubator**


## Доступные команды

### Чтобы запустить скрипт, введите в директории проекта:

```shell
yarn start
```

- команда запустит приложение в режиме разработки на порту: `3000`.

### Запустить тесты с помощью команды:
```shell
yarn test
```
